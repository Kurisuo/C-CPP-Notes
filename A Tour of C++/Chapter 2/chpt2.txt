Use pass-by-value (name) when I want to make a new independent copy of the original object without affecting the original
    - Avoid if using large object sets
Use pass-by-reference (reference) when I want to access the actual object at its address without worrying about poitner arithmatic 
    - A reference must always refer to a valid existing object, CANNOT be NULL
Use pass-by-pointer when dealing with something like dynamic memory
