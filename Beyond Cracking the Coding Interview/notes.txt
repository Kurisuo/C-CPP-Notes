
**Interview Checklist**

1 - Read Statement Twice
        - Work out examples outloud 
        - Ask for:
            - Edge cases
            - Input / output format
            - Scale Constraints

2 - Design the Algorithm 
        - Minimal Sketch of Naive Solution
        - Identify Bounds
        - Look for triggers

3 - Explain the Solution
        - Name & Justify

4 - Code the Solution
        - Main Logic
        - Helper Functions
        - STOP If I get confused

5 - Verify the Solution 
        - Top - Bottom Pass
        - Check Edge cases
        - Verify Analysis
